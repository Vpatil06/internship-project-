# HR Analytics -- Employee Attrition Analysis
HR Analytics -Predict Employee Attrition 
Objective:
Use analytics to identify key causes of employee attrition and build a predictive model to forecast future resignations.

## Tools Used:
- Python (Pandas, Seaborn, Sklearn)
- Power BI
- SHAP for explainable AI

## Project Workflow:

1. **Data Preprocessing & EDA**:
   - Department-wise attrition
   - Salary band analysis
   - Promotion trends

2. **Model Building**:
   - Classification using Decision Tree
   - Evaluated with accuracy score and confusion matrix

3. **Model Explainability**:
   - SHAP analysis to understand feature importance

4. **Visualization Dashboard**:
   - Power BI dashboard with attrition filters and insights

5. **Recommendations Report**:
   - PDF with actionable steps to prevent attrition

## Deliverables:
- `hr_attrition_data.csv`: Dataset used for analysis
- `attrition_prevention_suggestions.pdf`: Prevention strategy document
- Power BI dashboard file 
- Trained model and SHAP output
